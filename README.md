🏦 Banking Customer Intelligence Dashboard

A comprehensive 5-page Power BI Business Intelligence solution built using Power BI, Power Query, DAX, SQL concepts, and data visualization best practices to analyze customer demographics, financial performance, churn behavior, and customer risk.

📌 Project Overview

Banks generate enormous volumes of customer data every day. This project transforms raw banking data into interactive dashboards that help stakeholders monitor customer behavior, financial health, and churn trends to support data-driven decision-making.

The report contains five interactive dashboards designed for executives, business analysts, and relationship managers.

🚀 Dashboards
📊 1. Executive Dashboard

Provides a high-level overview of business KPIs including:

Total Customers
Active Customers
Churn Rate
Average Balance
Customer Distribution
Country Performance
👥 2. Customer Analytics

Analyzes customer demographics through:

Gender Distribution
Age Groups
Product Ownership
Active vs Inactive Customers
💰 3. Financial Analysis

Focuses on customer financial behavior.

Includes:

Average Balance
Estimated Salary
Credit Score Distribution
Balance by Country
Balance Category Distribution
📉 4. Churn Analysis

Provides insights into customer attrition.

Includes:

Churn KPIs
Churn by Country
Churn by Age Group
Churn by Customer Segment
Credit Score vs Churn
🎯 5. Customer Risk & Predictive Insights

Business intelligence dashboard for strategic decision making.

Includes:

Customer Segment Ranking
Balance Category Treemap
Customer Status by Salary
Country Performance Summary
Executive Insights
